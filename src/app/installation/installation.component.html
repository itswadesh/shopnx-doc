Installation Instructions

Requirements

We need Node (To create web server) + Git (Version Control System) + MongoDB (Database) + Python 2.7
Go ahead download and install them

Install Visual C++ Build Tools using the Default Install option.
Run the following commands (Required Once) [ Note: Second command is required only for Windows]

We need to start an instance of MongoDB. In my case it was sitting @ C:\Program Files\MongoDB\Server\3.2\bin\mongod.exe
